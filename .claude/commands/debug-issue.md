# Debug Issue (Apollo)

Rapidly debug issues with parallel file discovery and root cause analysis.

**Issue:** $ARGUMENTS

## Parallel Searches (5–10 simultaneous)

1. Exact error message
2. Mentioned function/class
3. Related file patterns
4. Similar test cases
5. Stack trace patterns
6. Config files
7. Recent changes in the area
8. Related dependencies

## Analysis

- Eliminate duplicates
- Identify file relationships
- Extract patterns
- Return prioritized findings

## Output

1. **Root Cause Hypothesis** — what's probably wrong
2. **Key Files** — most relevant files
3. **Reproduction Steps** — how to reproduce
4. **Suggested Fix** — solution direction
5. **Next Steps** — which agents to involve next
