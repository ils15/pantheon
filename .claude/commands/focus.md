# Focus — Pin Session Goal

Pin a session objective so all todos, delegation decisions, and verification steps stay aligned throughout this session.

**Goal:** $ARGUMENTS

## Steps

1. Confirm the goal is clear. If ambiguous, ask one clarifying question.
2. Format using the 4-component structure:

```
GOAL: <What to build>
SCOPE: <Which layers/modules are in scope>
SUCCESS CRITERIA: <How we know it's done>
OUT OF SCOPE: <What to explicitly exclude>
```

3. Write the pinned goal to `/memories/session/current-goal.md`.
4. Announce the pinned goal and confirm readiness to proceed.

## Alignment Rules (apply for the rest of this session)

- Every todo must relate to this goal
- Only invoke agents relevant to this goal
- Flag scope drift before proceeding
- Never change the goal silently — confirm with the user first
