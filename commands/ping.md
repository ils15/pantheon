# /ping — Agent Health Check

**What:** Pings all 17 Pantheon agents and returns a status table.
**Usage:** `/ping`
**Returns:** Table of all agents with status: ✅ available / ❌ unavailable / ⚠️ degraded
**Use when:** An agent seems unresponsive or you want to verify the system is healthy
