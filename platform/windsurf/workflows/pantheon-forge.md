# /pantheon-forge — Model Configuration

**What:** Switches the active model tier or sets a specific model ID.
**Usage:** `/pantheon-forge <preset|model-id>`
**Presets:** `fast` (cheap/quick), `default` (balanced), `premium` (deep reasoning)
**Example:** `/pantheon-forge premium` — switch to premium model tier
